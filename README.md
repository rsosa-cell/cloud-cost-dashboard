
# ☁️ Cloud Cost Intelligence Dashboard

A real-time Google Cloud billing analytics dashboard built with React and TypeScript.

Users can monitor cloud spending, detect anomalies, view cost breakdowns, and receive AI-powered optimization insights.

[▶ Watch Demo](https://www.youtube.com/watch?v=yxVV3M3BYf0)

---

## 🚀 Features

### 📊 Real-time Cloud Cost Analytics
- Live Google Cloud billing data
- Auto-refresh every 10 seconds
- Daily cost tracking

### 📈 Interactive Visualizations
- Line chart (cost over time)
- Bar chart (cost per service)
- Pie chart (cost distribution)

### ⚠️ Smart Cost Monitoring
- Budget exceeded alerts
- Cost trend indicators (↑ / ↓)
- Top 3 most expensive services
- Anomaly detection (spike alerts)

### 🧠 AI-Powered Insights
- Cost summaries
- Optimization suggestions
- Spending behavior analysis

### 🔮 Predictive Analytics
- Monthly cost projections
- Future spend estimation
- Savings recommendations

### 📁 Export Tools
- Download cost data as CSV

---

## 🧠 Project Goal

To transform raw Google Cloud billing data into clear, actionable financial insights using real-time analytics and AI.
